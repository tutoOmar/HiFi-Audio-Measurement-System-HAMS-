# HiFi-Audio-Measurement-System-HAMS-
Sistema de medición de audio de alta fidelidad basado en el ATSAMD21G18A, el ADC PCM4201 y el DAC PCM5100A. Permite adquirir, analizar y visualizar ruido y distorsión mediante I2S, reloj externo y módulos en LabVIEW. Busca ofrecer una solución abierta, precisa y de bajo costo para entornos académicos.
